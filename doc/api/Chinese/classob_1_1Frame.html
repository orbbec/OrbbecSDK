◆ globalTimeStampUs()

uint64_t ob::Frame::globalTimeStampUs ( )

获取帧的全局时间戳（以微秒为单位）。

全局时间戳是帧在设备上获取当前帧的时间点，并转换为主机时钟域。转换过程基于设备时间戳，可以消除设备时钟漂移。

注意
只有某些设备支持获取全局时间戳。如果设备不支持，此函数将返回0。可以通过 Device::isGlobalTimeStampSupported() 检查设备是否支持获取全局时间戳。
返回
uint64_t 返回帧全局时间戳（以微秒为单位）
