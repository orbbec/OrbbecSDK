通过 UID 创建设备

在 Linux 平台，设备的 uid 由 bus-port-dev 组成，例如 1-1.2-1。但是 SDK 内部会去掉 dev 号只保留 bus-port 当作 uid 来创建设备，例如 1-1.2，这样我们就可以创建一个连接到指定 USB 端口的设备。相应地，用户也可以直接传入 bus-port 当作 uid 来创建设备。

注意
如果设备有在其他地方被获取创建，重复获取将会返回错误
参数
[in]list设备列表对象。
[in]uid要创建的设备的 UID。
[out]error记录错误信息。
返回
ob_device * 创建的设备。
