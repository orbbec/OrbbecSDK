uint16_t OBDeviceSyncConfig::mcuTriggerFrequency

MCU 触发频率，用于 MCU 主模式下，MCU 触发信号输出频率配置，单位：Hz.

该配置会直接影响 Sensor 的图像输出帧率，即也可以认为单位为：FPS （frame pre second）

注意
仅当同步模式配置为 OB_SYNC_MODE_PRIMARY_MCU_TRIGGER 时无效
