void ob::Context::enableNetDeviceEnumeration ( bool  enable)

使能或禁用网络设备枚举。

使能后，网络设备会自动被发现，并且可以通过 queryDeviceList 获取。默认状态可以在配置文件中设置。

注意
网络设备枚举通过 gvcp 协议，如果设备和主机不在同一个子网，会被发现但是无法连接。
参数
[in]enabletrue 使能，false 禁用
