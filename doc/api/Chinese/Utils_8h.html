[in]data外部分配用于保存xy表的内存。在初始化过程中，外部分配的大小为'data_size'，例如，data_size = 1920 * 1080 * 2*sizeof(float)（1920 * 1080代表图像分辨率，2代表两个LUT，一个用于x坐标，一个用于y坐标）.
[in]data_size输入data的内存大小
