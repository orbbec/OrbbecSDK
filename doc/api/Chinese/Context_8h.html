bool  enable,
ob_error **  error 
)

启用或禁用网络设备枚举

启用后，网络设备将自动被发现，并且可以通过 ob_query_device_list 获取。默认状态可以在配置文件中设置。

注意
网络设备枚举通过 gvcp 协议，如果设备和主机不在同一子网，将被发现但无法连接。
参数
[in]context指向上下文对象的指针
[in]enabletrue 启用，false 禁用
[out]error指向错误对象的指针，如果发生错误，将填充该对象
