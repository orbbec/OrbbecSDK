float ob::PointsFrame::getPositionValueScale ( )

获取点云帧的点坐标值缩放系数，点坐标值乘以缩放系数后，可以得到单位为毫米的坐标值； 如 scale=0.1, 某个点的 x 坐标值为 x=10000， 则表示实际 x 坐标 value = x*scale = 10000*0.1=1000mm。
