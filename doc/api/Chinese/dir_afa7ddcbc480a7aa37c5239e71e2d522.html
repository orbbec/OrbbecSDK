h 目录参考

文件

 Context.h
 context 是描述 SDK 的 runtime 一个管理类，负责 SDK 的资源申请与释放。 context 具备多设备的管理能力，负责枚举设备，监听设备回调，启用多设备同步等功能。
 
 Device.h
 设备相关功能，包括获取和创建设备、设置和获取设备属性、获取传感器等操作
 
 Error.h
 处理错误的函数，主要用于获取错误信息。
 
 Filter.h
 SDK 的处理单元, 可以执行点云生成、格式转换等功能。
 
 Frame.h
 帧相关函数, 主要用于获取帧数据和帧信息
 
 MultipleDevices.h
 该文件包含多个设备相关的 API 开关，用于控制多个设备之间的同步以及单个设备内不同传感器之间的同步
 
 ObTypes.h
 提供 SDK 中常用的结构体，枚举常量定义。
 
 Pipeline.h
 SDK 的高级 API，可以快速实现开关流，帧同步，软件滤波等功能，适用于应用，算法重点关于 rgbd 数据流场景。如果对实时性或需要单独 处理的同步，对齐的场景。请使用 Device 的 Lower API 的接口。
 
 Property.h
 控制命令属性列表维护
 
 RecordPlayback.h
 用于录制和回放功能
 
 Sensor.h
 传感器相关函数，用于获取流配置，开关流，设置及获取传感器属性等操作
 
 StreamProfile.h
 流配置相关函数，用于获取流的宽、高、帧率及格式等信息
 
 Version.h
 获取 SDK 版本号相关信息函数
