int ob_multi_device_sync_config::trigger2ImageDelayUs

在接收到捕获命令或触发信号之后的图像捕获的延迟时间（以微秒为单位）

深度图像和彩色图像是作为产品设计同步捕获的，并且不能改变深度图像和色彩图像之间的延迟

注意
此参数仅对某些型号有效，以替换 depthDelayUs 和 colorDelayUs ，请参阅产品手册了解详细信息

在文件 ObTypes.h 第 1109 行定义.

bool ob_multi_device_sync_config::triggerOutEnable

触发信号输出启用标志

在触发信号输出被使能之后，当接收到捕获命令或触发信号时，触发信号将被输出。用户可以调整 triggerOutDelayUs 输出的触发信号的延迟时间

注意
对于某些型号，触发信号输出始终处于启用状态，在某些模式下无法禁用
如果设备处于 OB_MULTI_DEVICE_SYNC_MODE_FREE_RUN 或 OB_MULTI_DEVICE_SYNC_MODE_STANDALONE 模式，则触发信号输出始终被禁用。将此参数设置为 true 将不会生效
